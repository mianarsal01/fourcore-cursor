# FourCore Cursor

Shopify app for adding custom cursors to storefronts using a lightweight theme app embed.

## Local development

```bash
npm install
npm run shopify app dev
```

## Key paths

- Admin UI: `app/routes/app._index.tsx`
- Theme app embed: `extensions/cursor-extension/blocks/cursor_embed.liquid`
- Cursor assets: `extensions/cursor-extension/assets/`
- PNG generator: `scripts/generate-cursors.mjs`
